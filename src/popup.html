EMSCharts Filler

This extension allows you to set defaults for your EMSCharts PCRs.

If this is your first time using this extension, right click on this icon and choose "Options". From there, you'll want to look at the categories and fill in your default values.

To report an issue, request a new feature, or complain about how much it sucks, consider joining the Google Group for this extension, which is located at: http://groups.google.com/chartfiller-support

Source code is available upon request.
