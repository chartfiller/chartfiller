EMSCharts Filler

This extension allows you to set defaults for your EMSCharts PCRs.

I'm no longer active in EMS and don't have access to continue to support/test/debug certain issues. If you're interested in maintaining/contributing/testing, email me at chartfiller@cmattoon.com

If this is your first time using this extension, right click on this icon and choose "Options". From there, you'll want to look at the categories and fill in your default values.

To report an issue, request a new feature, or complain about how much it sucks, visit the issues tab on GitHub

Source code is available at http://cmattoon.github.io/chartfiller/.
